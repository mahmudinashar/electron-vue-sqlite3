#### Build Setup

``` bash

# clone project
git clone https://github.com/mahmudinashar/electron-vue-sqlite3.git

# enter project folder
cd electron-vue-sqlite3

# install dependencies
yarn or npm install

# running project in development
npm run dev

# build project
npm run build

# lint all JS/Vue component files in `src/`
npm run lint:fix

```
